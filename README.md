## 概要
空間をスキャンして3Dオブジェクトにし、STYLYにアップロードして、STYLYでその空間を自由に遊べるようにしました。

## やり方
1. Scaniverseで部屋をスキャンして、3Dオブジェクト(.fbx)を作成。
1. そのfbxファイルをUnityにインポートして、色をつけてSTYLYにアップロード
1. STYLYでモディファイアーを追加していい感じに
1. 完成！
